## Artist Profile

Comedy group from Britain.
Formed 1969–1983, 1989, 1998–1999, 2002, 2013–2014.

Members:
Graham Chapman
John Cleese
Terry Gilliam
Eric Idle
Terry Jones
Michael Palin
